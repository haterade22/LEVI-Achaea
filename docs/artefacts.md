## Notes

- **Multiple copies**: Some artefacts (vials, earrings, lodestones) you have multiples of
- **Naming**: Items in parentheses show your custom name vs default name
- **IDs**: Use `ARTEFACT <id>` or `PROBE <id>` in-game for full details
- **Upgrades**: Many artefacts can be upgraded (L1 → L2 → L3)
